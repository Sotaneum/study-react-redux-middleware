# study-react-redux-middleware

## 프로젝트 세팅하기

- 라이브러리 설치하기

```bash
npm install --registry https://registry.npmjs.com
```

## 프로젝트 시작하기

```bash
npm run start
```

## 프로젝트 빌드하기

```bash
npm run build
```

## 프로젝트 배포하기

```bash
npm run deploy
```
